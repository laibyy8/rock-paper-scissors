# ✊✋✌️ Rock Paper Scissors

A clean, animated Rock Paper Scissors game built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🎮 Live Demo

> Open `index.html` in your browser and start playing instantly!

---

## ✨ Features

- 🤚 **Custom SVG hand illustrations** for Rock, Paper & Scissors
- 🎯 **Animated score counter** with bump effect on every point
- 💬 **Result messages** with win / lose / draw color themes
- ⏳ **Thinking animation** while CPU picks its move
- 🔄 **Reset button** to start a fresh game
- 📱 **Responsive design** — works on mobile and desktop
- 🚀 **Zero dependencies** — just open and play

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. That's it — no install, no server needed!

```bash
git clone https://github.com/YOUR_USERNAME/rock-paper-scissors.git
cd rock-paper-scissors
# Open index.html in your browser
```

---

## 🕹️ How to Play

1. Click **Rock**, **Scissors**, or **Paper**
2. The CPU picks a random move
3. Winner is decided by classic rules:
   - Rock crushes Scissors
   - Scissors cuts Paper
   - Paper covers Rock
4. Score is tracked across rounds
5. Hit **Reset** to start over
